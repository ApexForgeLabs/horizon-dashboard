## 🚀 Overview

**Horizon** is a sleek, responsive admin dashboard for data-heavy apps. Built on Next.js 15 App Router with shadcn/ui components and real-time charts.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Modern UI** | shadcn/ui + Tailwind CSS |
| 📊 **Charts** | Interactive charts with Recharts |
| 🌙 **Dark Mode** | System-aware theme switching |
| 📱 **Responsive** | Mobile-first layout |
| 🔐 **Auth Ready** | NextAuth.js with multiple providers |
| ⚡ **Server Components** | Optimized rendering with RSC |

## ⚡ Quick Start

```bash
git clone https://github.com/ApexForgeLabs/horizon-dashboard.git
cd horizon-dashboard
pnpm install
pnpm dev
```

Open `http://localhost:3000`
